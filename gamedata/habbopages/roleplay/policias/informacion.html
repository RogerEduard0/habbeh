TUTORIAL: Cómo ser policia
1- Activa tu empleo de policia

Debes ubicar este furni en la sala y ponerte encima de él para activar tu empleo como policia, colocarte tu uniforme y recibir tu primer arma.
Dato: Si perdiste tu arma puedes volver a pisar este furni para obtenerla nuevamente.
2- Usa el comando :expediente usuario

Teniendo tu empleo de policia activo podrás usar el comando de :expediente usuario para ver los expedientes de cada uno de los usuarios. De esta manera podrás ver sus delitos y podrás seguirlos para arrestarlo.
Importante: Siendo policia recibirás alertas cuando los usuarios cometan delitos, podrás presionar las alertas para seguirlos. ¡Debes estar atento!
3- Utiliza el comando :arrestar usuario

Puedes utilizar el comando :arrestar usuario para enviar al usuario a prisión. Ojo, esto siempre y cuando el usuario tenga arrestos pendientes.

Si el usuario no tiene arrestos pendientes, te mostraremos su expediente.
Nota: Una vez envies a un usuario a prisión, quedarás sin arma...
4- Acercate a la cartelera, reporta tu arresto para incrementar tu sueldo

Debes acercarte a la cartelera informativa y clickearla dos veces para comenzar a armar tu reporte, esto lo harás en un corto periodo de tiempo. (deberás mantenerte cerca de la cartelera, si te alejas se cancelará el proceso).

Una vez finalizado tu reporte, volverás a obtener tu arma y tu sueldo incrementará.
Nota: El empleo funciona de esta manera para evitar el abuso de poder y mejorar la experiencia de juego de todos los roles.
5- Desactiva tu empleo como policia y recibe tu paga.

Si quieres recibir tu paga, deberás ponerte encima de este furni, de esta manera desactivarás tu empleo y recibirás correctamente tu sueldo acumulado.
